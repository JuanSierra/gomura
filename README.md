# Gomura
Simple web platform for playing Stop game on multiplayer

## Requirements
####MongoDB
1.  Install it from [mongodb.org](https://www.mongodb.org/downloads) 
2.  Run mongod from a command line

## Installation

```sh
git clone https://github.com/JuanSierra/gomura.git
cd gomura
npm install
```

## Usage

```sh
npm start
```
